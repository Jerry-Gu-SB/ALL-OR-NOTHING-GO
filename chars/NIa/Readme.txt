Kung Fu Man is a Sample Character Template that works for M.U.G.E.N

--------
Moveset:
--------
X = Light Punch
Y = Medium Punch
Z = Heavy Punch
A = Light Kick
B = Medium Kick
C = Heavy Kick
S = Taunt

--------------
Special Moves:
--------------

Kung Fu Palm: D, DF, F, x/y/z
$D, $U = Super Jump

------------
Hyper Moves:
------------
Triple Kung Fu Palm (requires 1000 power): D, DF, F, x+y
